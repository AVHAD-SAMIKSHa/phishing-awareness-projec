# Phishing Awareness Simulation Project

## Introduction
This project demonstrates a phishing awareness simulation using social engineering techniques. The aim is to understand how phishing attacks work and educate users about cybersecurity threats.

## Objectives
- Understand phishing techniques
- Simulate a phishing email scenario
- Analyze user behavior when receiving phishing emails
- Increase awareness about phishing attacks

## Tools Used
- HTML
- GitHub
- Gmail
- Google Forms

## Project Workflow
1. A phishing email was created and sent to participants.
2. The email contained a link to a simulated login page.
3. Users entered information on the login page.
4. The page redirected users to a survey form.
5. Responses were collected and analyzed.

## Preventive Measures
- Verify sender email addresses
- Avoid clicking suspicious links
- Enable multi-factor authentication
- Report phishing emails

## Conclusion
This project demonstrates how phishing attacks exploit human behavior and highlights the importance of cybersecurity awareness.
